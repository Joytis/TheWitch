# Docs Index

## Art
- **[Art Tracker](art-tracker.html)** — static, team-facing status page for all art assets (Cards / Potions / Relics / Familiar Pets / Character & UI). Regenerated by `node Docs/card-data/regen.js` (or `/regen`); served locally or via GitHub Pages.
  - Data sources: [art-tracker/assets.json](art-tracker/assets.json) (non-card assets), [art-tracker/card-briefs.json](art-tracker/card-briefs.json) (per-card artist + brief).
  - Authoring view: run `Docs/card-designs.cmd` → edit Artist / Art Brief / Art-Final on the Witch tab; the tracker auto-regenerates.
